### 1. Time Budget System

**Specification:**
```yaml

  quick:
    time_budget: 60  # seconds (MUST - strict requirement)
    delegation: false
    output: "5-7 ideas, quick wins"

    delegation: optional
    output: "comprehensive with options"

```
