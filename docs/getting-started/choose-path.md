### Common Tasks

- **Check code quality:** `/craft:code:lint` or `/craft:code:ci-local`
- **Update documentation:** `/craft:docs:update` or `/craft:docs:sync`
- **Create a site:** `/craft:site:build` ‚Üí `/craft:site:deploy`
- **Manage git workflow:** `/craft:git:worktree` ‚Üí `/craft:git:branch`
- **Smart routing:** `/craft:do <task>` (auto-detects the right command)
